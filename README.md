📊 DonateKart Excel Dashboard Analysis
📌 Description
This project is an interactive Excel dashboard analyzing yearly donation data from DonateKart. It provides insights into total donations collected, number of donors, and overall campaign performance, helping identify trends and successful fundraising strategies.
🎯 Objectives
Analyze annual donation trends
Track total number of donors
Evaluate campaign performance
Identify campaigns that met or exceeded targets
Provide actionable insights for better decision-making
📂 Dataset
Source: Sample/Provided dataset (DonateKart platform data)
Time Period: Full year data
Key Fields:
Donation Amount
Donor Details
Campaign Name
Target Amount
Amount Achieved
Date/Time
✨ Dashboard Features
Total Donations Overview
Total Number of Donors
Campaign-wise Performance Analysis
Target vs Achieved Comparison
Monthly Donation Trends
Interactive Filters (Slicers) for better analysis
🛠️ Tools & Technologies Used
Microsoft Excel
Pivot Tables
Pivot Charts
Data Cleaning & Transformation
Slicers & Filters
🚀 How to Use
Download the Excel file from this repository
Open it using Microsoft Excel
Use slicers/filters to explore different insights
Analyze trends and campaign performance
📈 Key Insights
Top-performing campaigns contributed the highest share of total donations
Certain months showed peak donation activity indicating seasonal trends
A significant portion of campaigns successfully met their targets
Donor participation varied across campaigns, highlighting engagement patterns
